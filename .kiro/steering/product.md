# Product Overview

Noraneko は、モダンなフルスタック技術を使用したTODOリスト管理システムです。開発者が学習・参考にできる実践的なモノレポ構成のWebアプリケーションです。

## Core Capabilities

- **タスク管理** - TODOの作成、更新、削除、完了状態の切り替え
- **リアルタイム同期** - フロントエンドとバックエンドの状態管理
- **型安全な開発** - TypeScriptによるエンドツーエンドの型安全性
- **モダンUI** - PrimeReactを使用したレスポンシブなインターフェース

## Target Use Cases

- **学習プロジェクト** - モノレポ、React、Express、Prismaの実践的な学習
- **プロトタイプベース** - 新機能の実験と検証のための基盤
- **アーキテクチャ参考** - フロントエンドとバックエンドの分離、状態管理パターンの参考実装

## Value Proposition

- **モノレポ構成** - Yarn Workspacesによる効率的なコード共有とパッケージ管理
- **型安全性** - フロントエンドからバックエンド、データベースまで一貫したTypeScript活用
- **ベストプラクティス** - Redux Saga、Prisma ORM、ESLint/Prettierなどの業界標準ツールの統合
